# HR Analytics Dashboard

## Project Overview
The HR Analytics Dashboard is designed to analyze employee data and provide insights into workforce trends, employee attrition, job satisfaction, and overall HR performance. This dashboard helps HR teams make data-driven decisions to improve employee retention and organizational effectiveness.

## Key Performance Indicators (KPIs)

### Total Employees
- Displays the total number of employees in the organization.

### Attrition Count
- Shows the total number of employees who left the company.

### Attrition Rate
- Calculates the percentage of employees who left the organization.
- Formula:

  Attrition Rate = (Attrition Count / Total Employees) × 100

### Active Employees
- Displays the number of currently active employees.

### Average Age
- Shows the average age of employees in the organization.

## Dashboard Features

### Attrition Analysis
- Analyze employee attrition across departments.
- Identify departments with the highest turnover.

### Job Satisfaction Analysis
- Evaluate employee satisfaction levels.
- Compare satisfaction across different job roles.

### Age Group Analysis
- Understand workforce distribution by age group.
- Identify trends among different employee demographics.

### Education Field Analysis
- Analyze employee distribution based on educational background.

### Gender Analysis
- Compare workforce composition by gender.
